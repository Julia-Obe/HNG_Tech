# HNG Tech Internship – Data Analytics Track

## Project Overview

This repository contains my work completed during the HNG Tech Internship (Data Analytics Track). Throughout the internship, I worked on real-world datasets to develop practical skills in data cleaning, data transformation, SQL querying, exploratory data analysis, short title creation for SEO, and business intelligence.

The projects focused on transforming raw data into meaningful insights using industry-standard data analytics tools and techniques.

---

## Internship Objectives

- Clean and prepare raw datasets for analysis.
- Perform exploratory data analysis (EDA).
- Query and analyze data using SQL.
- Apply analytical thinking to solve business problems.
